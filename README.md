# 📝 To do List (React) 

## ✨ 주요 기능
- ➕ **할 일 추가** : 새로운 할 일을 입력하고 추가할 수 있어요.
- 📋 **할 일 목록 표시** : 등록된 할 일들을 한눈에 볼 수 있어요.
- ✅ **할 일 완료 토글** : 할 일을 클릭하면 완료/미완료 상태를 바꿀 수 있어요.
- 🗑️ **할 일 삭제** : 필요 없는 할 일은 삭제할 수 있어요.
